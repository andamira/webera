# Contributor Code of Conduct

> en

```
Since we are all human, conscious and intelligent beings,
with the ability to understand and learn from each other,
let's keep doing that: honestly working on the source code,
focusing on the job that needs to be done, being excellent.
```

> es

```
Siendo todos seres humanos, conscientes e inteligentes,
con capacidad de entender y aprender los unos de otros,
sigamos así: trabajando con honestidad el código fuente,
centrándonos en el trabajo por hacer, que sea excelente.
```
